# object-detection-webapp
Welcome to our Oldschool WebApp! This is a simple web application built using old-school HTML, CSS, and JavaScript. It's a blast from the past, showcasing the fundamentals of web development before all the fancy frameworks and libraries came into play.

## Features
1. Pure HTML: The structure of the web pages is built using plain HTML without any frameworks.
2. CSS Styling: The styles are applied using traditional CSS for a simple yet effective design.
3. Vanilla JavaScript: Interactivity is achieved through pure JavaScript, demonstrating basic DOM manipulation and event handling.

## Usage
1. Start the backend server >>> https://github.com/aravind-tronix/object-detection-web-client
2. To use this web application, simply clone the repository and open the index.html file in your favorite web browser.
3. `https://github.com/aravind-tronix/object-detection-webapp.git`
4. `cd object-detection-webapp`
5. Then, open the index.html file in your browser.

## Structure
1. index.html: The main HTML file containing the structure of the web pages.
2. .css: The CSS file for styling the web pages.
3. .js: The JavaScript file for adding interactivity to the web pages.
