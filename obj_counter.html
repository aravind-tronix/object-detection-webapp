This is a demonstration how machine learning algorithms are used to detect objects.With help of Darknet yolo object detection ,here is a webapp which can able to detect/count objects of several kinds. This model is customly trained and a custom model is created which contains all detected assets. Further, a API is been developed which can be used on any platforms to scan contents before uploading and can be used to block if any sensitive contents present in the image.
